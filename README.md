## 📕️ Como baixar o projeto

```bash
    #Clonar o repositório
    $ git clone https://github.com/Liliamsn

    #Entrar no diretório
    $ cd alurachat

    #Instalar o yarn globalmente
    $ npm install --global yarn

    #Inicializar o projeto
    $ yarn run dev
```
